# Paper Implementations

Welcome to my repository where I share my personal implementations of famous Machine Learning and Deep Learning research papers. The goal is to break down complex architectures and build them from scratch for better understanding.

## Papers Covered

Here is the list of papers currently implemented in this repository. I will keep this list updated as I add new ones!

| Paper | Implementation | Key Concepts / Description |
|-------|----------------|----------------------------|
| [Attention Is All You Need](https://arxiv.org/abs/1706.03762) | [Transformer Notebook](./Attention_is_all_you_need/main.ipynb) | The original Transformer architecture covering Self-Attention, Multi-Head Attention, and Encoder-Decoder blocks. |
| [MapReduce: Simplified Data Processing on Large Clusters](https://research.google/pubs/pub62/) | [C++ Word Count Example](./mapreduce/main.cpp) | A beginner-friendly C++ implementation of the classic MapReduce flow: Map, Shuffle/Group, and Reduce using a word-count example. |
